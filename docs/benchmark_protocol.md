# Benchmark Protocol

- Track A measures semantic spoken QA at scale.
- Track B measures audio-essential samples with transcript-fixed label flips.
- Required conditions: Text-Oracle, Text-ASR, Speech-Natural, Speech-Lexicalized, optional Speech-Revoiced.
- Required metrics: ASR Loss, Modality Penalty, Incremental Audio Value, Prosody Sensitivity.
- Never make text-sufficient or audio-essential claims from inconclusive slices.
